# Student Grade Tracker (Part 1)

## Overview
This project is a command-line based Student Grade Tracker built using Python basics.

## Features
- Cleans and formats raw student data
- Validates student names
- Calculates grades, total, and average marks
- Allows user to add new subjects dynamically
- Generates class performance report
- Performs string operations on an essay

## How to Run
1. Open terminal
2. Run: python part1_grade_tracker.py
3. Follow the instructions on screen

## Concepts Used
- Loops (for, while)
- Conditionals (if-else)
- Lists & dictionaries
- String manipulation
- Functions
